# PIP Coverage Advisor

A personalized Personal Injury Protection (PIP) coverage recommendation tool.

## Features

- Interactive questionnaire to assess your coverage needs
- Personalized recommendations based on your situation
- Coverage calculations for Work Loss Benefits
- Clear explanations of different PIP benefit types
- Mobile-responsive design

## Local Development

```bash
npm install
npm run dev
```

## Deployment to Vercel

### Quick Deploy (Recommended)

1. Push this repository to GitHub
2. Go to [Vercel](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Vercel will auto-detect the Vite configuration
6. Click "Deploy"

### Manual Configuration

If needed, configure these settings in Vercel:
- **Framework Preset**: Vite
- **Build Command**: `npm run build`
- **Output Directory**: `dist`

## Technology Stack

- React 18
- Vite
- Tailwind CSS (via CDN in build)
- Lucide React (icons)
